# Smart Attendance System

Welcome to the Smart Attendance System project. This is a group project  aims to simplify and automate the process of taking attendance using facial recognition technology. The system leverages Python, OpenCV, and face_recognition libraries to identify and verify individuals.

## Features

- Automatic face detection and recognition
- Real-time attendance logging
- Easy integration with existing systems
- Scalable and customizable

## Technologies Used

- Python
- OpenCV (cv2)
- face_recognition library
- OS library

## Installation

To get started with the Smart Attendance System, follow these steps:

```bash
git clone https://github.com/your-repo/smart-attendance-system.git
cd smart-attendance-system
pip install -r requirements.txt
python main.py
